About Vel Tech Rangarajan Dr.Sagunthala R&D Institute of Science and Technology Portal

A production-style academic management portal built on Java 17 and Spring Boot.

Mission

Deliver transparent, secure, and efficient digital workflows for students, faculty, and administrators using a clean full-stack architecture.

Architecture

MVC pattern with layered packages: controller, service, repository, entity, config, and dto. JWT security protects API modules by role.

Security

BCrypt password hashing, stateless JWT authentication, method-level role authorization, and centralized exception handling.

Modules

Attendance tracking, assignment upload and evaluation, role dashboards, and admin controls for users and course management.
